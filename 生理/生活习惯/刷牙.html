# 如何刷牙

# 方法

巴氏刷牙法

  • 手持牙刷刷柄，先将刷头放置于口腔内一侧的后牙牙颈部，刷毛与牙长轴大约呈45°角，刷毛指向牙根方向（上颌牙向上，下颌牙向下），轻微加压，使刷毛部分进入牙龈沟内，部分置于牙龈上；
  • 以2~3颗牙为一组开始刷牙，用短距离水平颤动的往返动作在同一个部位至少刷10次，然后将牙刷向牙冠方向转动，继续拂刷牙齿的唇（颊）舌（腭）面；
  • 刷完第一个部位之后，将牙刷移至下一组2~3颗牙的位置重新放置，注意与第一个部位保持有重叠的区域，继续进行下一个部位的刷牙；
  • 刷上前牙舌面时,将刷头竖放在牙面上，使前部刷毛接触龈缘，自上而下拂刷。刷下前牙舌面时，自下而上拂刷；
  • 刷咬合面时，刷毛指向咬合面，稍用力作前后短距离来回刷。

# 刷牙时间

时间约为3-5分钟

# 频率

每天早晚各一次

  • 起床之后,吃早饭之前（早饭之后也可以，无固定结论）
  • 睡觉之间

每一餐饭后也都可以刷牙，但是一般都不具备空间时间条件，所以推荐漱口 漱口可以用漱口水也可以用清水 漱口要闭上嘴巴，使水流冲洗牙缝；如果是张着嘴巴，是达不到清晰的效果的

# 牙刷

  • 牙刷要选软毛的
  • 牙刷每三个月更换一次

# 电动牙刷

牙周不好不要用电动牙刷

# 牙膏

关键词：含氟，水合硅石

适用于“黄牙”，牙齿有结石，由于茶渍，烟渍，咖啡渍等

  • xx磷酸钠：化学清洁剂
  • 水合硅石：物理磨牙剂

蛀牙

  • 含氟量%0.14以上：预防修复蛀牙

牙齿敏感，口臭

  • 含有硝酸钾，氯化锶，磷硅酸钠钙等抗敏成分

# 牙线

提倡用牙线代替牙签

# 洗牙

每年至少洗一次牙

很多人认为洗牙只是对牙齿进行清洁，所以多刷牙可以代替洗牙，这是不对的。洗牙是用专业医疗器械去除牙石、菌斑和色素等，并对牙表面进行抛光或喷砂，使牙齿表面恢复光泽。而刷牙只能去掉食物残渣等，一旦形成牙结石，就只能通过洗牙的方式来去除。洗牙之后有人会觉得牙齿敏感、发酸，这是牙结石被去掉后，牙根暴露出来了，不用担心，过几天就会恢复正常。
