# 艾滋病 AIDS

艾滋病，全称是“获得性免疫缺陷综合征” （英文名：Acquired Immunodeficiency Syndrome，缩写AIDS） 艾滋病（疾病）_百度百科 (baidu.com) (opens new window)

它是由艾滋病病毒即人类免疫缺陷病毒（HIV）引起的一种病死率极高的恶性传染病。

HIV病毒侵入人体，能破坏人体的免疫系统，令感染者逐渐丧失对各种疾病的抵抗能力，最后导致死亡。

艾滋病的结果就是使人体丧失免疫能力，不是导致死亡的直接原因；导致死亡的直接原因，一般是由于人体缺乏免疫能力的患的其他并发病

目前还没有疫苗可以预防

# HIV 病毒

引发艾滋病的HIV是一种正链RNA逆转录病毒。

HIV以人体免疫系统中的CD4+ T细胞 (opens new window)（辅助性T细胞，Th）为攻击目标。在急性感染阶段，HIV通过gp41 (opens new window)与gp120 (opens new window)组成的复合体侵入CD4+ T细胞，导致CD4+ T细胞的数量开始下降。侦测到HIV存在的免疫系统很快会作出反应，杀死大部分的HIV，令CD4+ T细胞的数量几乎恢复到正常水平。但HIV病毒通过高频率的突变 (opens new window)，最后能逃过免疫系统的追捕，转为潜伏状态。在HIV处于潜伏状态的潜伏期，CD4+ T细胞的数量持续下降。最后，当CD4+ T细胞数量下降到极低水平、HIV病毒上升到一定水平后，艾滋病也随之进入症状期。没有CD4+ T细胞辅助的后天免疫系统几乎失灵，免疫系统将难以杀灭侵入机体的病原体以及体内发生癌变的细胞，患者很快就会死于感染或恶性肿瘤

# 艾滋病的历史

  • 约1920年，非洲人感染来自黑猩猩的HIV
  • 1981年，美国登载了5例艾滋病病人的病例报告，这是世界上第一次有关艾滋病的正式记载。
  • 1982年，这种疾病被命名为"艾滋病"。
  • 1985年，一位到中国旅游的外籍人士患病，这是中国第一次发现艾滋病病例。

研究认为，艾滋病起源于非洲，后由移民带入美国。后蔓延至全世界。 关于艾滋病来源的猜想？

  • 猜想一：野生的猩猩攻击和撕咬人群，造成病毒的感染和扩散
  • 猜想二：非洲偏远地区的人捕猎并食用黑猩猩，但因为卫生条件极端低下，有的都不经过完全煮熟，导致猩猩身上的病毒传染给人体。
  • 猜想三：艾滋病被认为是非洲土著人与猩猩的不安全性行为导致的，这是人兽不伦行为的一种恶果。

# 艾滋病的症状

艾滋病毒感染症状视感染阶段而异。 一般将艾滋病分为：急性感染期、潜伏期、症状期,以及典型AIDS发病期

# 急性感染期

在感染后的最初几个月里，该疾病更容易传播，但许多人直到后期才意识到自身状况。

在感染后的最初几周内，人们可能不会出现症状，也可能出现流感样疾病。艾滋病急性期症状至少高危行为后的第三天才会有部分出现，而这种症状不会发生在高危的10周以后，绝大部分人出现急性症状的时间是在10-14天，持续时间1到2周，而需要知道的是有50%以上的艾滋病感染者初期是不出现急性期症状的。

艾滋病感染者出现初期症状的比例：

  1. 发热（80%）
  2. 嗜睡和全身不适（70%）
  3. 肌肉痛和关节痛（50 - 70%）
  4. 淋巴结肿大（40 - 70%）
  5. 盗汗（50%）
  6. 胃肠炎（50 - 70%）
  7. 腹泻30%）
  8. 口腔溃疡（10 - 30%）
  9. 神经性头痛（40 - 70%）
  10. 皮疹（40 - 80%）
  11. 生殖器溃疡（5 - 15%）
  12. 血小板减少（45%）
  13. 白细胞减少（40%）
  14. 转氨酶上升（20%）

# 潜伏期

HIV急性感染阶段后，艾滋病进入潜伏期，这一时期感染者仍然不会出现明显症状。 艾滋病的潜伏期长度介于数月到20年之间。 根据统计，艾滋病潜伏期的平均长度大约是10年

一般将病程处于急性感染期与潜伏期的感染者称为“HIV携带者”，而将进入症状期（发病期）的感染者称为“艾滋病人”。

# 症状期

最后，HIV感染者的免疫系统几乎被HIV摧毁，艾滋病进入症状期（发病期）。几乎失去所有免疫力的患者将很快死于感染或恶性肿瘤

感染逐渐削弱免疫系统。这可能会导致其他体征和症状：

  • 淋巴结肿大
  • 体重减轻
  • 发烧
  • 腹泻
  • 咳嗽

如果不进行治疗，艾滋病毒感染者也可能患上严重疾病：

  • 结核病（TB）
  • 隐球菌性脑膜炎
  • 重度细菌感染
  • 淋巴瘤和卡波西肉瘤等癌症

艾滋病毒会导致其他感染的恶化，例如丙型肝炎、乙型肝炎和猴痘。

# 艾滋病的检测

# 窗口期

从艾滋病病毒进入人体到血液中产生足够量的、能用检测方法查出艾滋病病毒抗体之间的这段时期，称为窗口期。

在窗口期虽测不到艾滋病病毒抗体，但体内已有艾滋病病毒，可以通过HIV核酸检测查到，因此处于窗口期的感染者同样具有传染性。

世界卫生组织WHO明确表示艾滋病窗口期（WindowPeriod）为14～21天

这也就意味着距离上一次高危性行为14~21天后，才能检测出来HIV，在这期间内是检测不出来的，但是仍然具有传播风险。

在之前的检测技术中，窗口期被认为是3个月；随着检测技术的提高，可缩短至3到4周

# 去医院

1.去医院检测 医院或疾控中心都可以检测艾滋病，去医院需要挂皮肤科或传染科，目前医院的检测方法一般有三种：

（1）RNA核酸检测： 窗口期7天，费用100-1000元不等，检测结果1-5个工作日出具。 （2）四代P24抗原抗体双测： 窗口期14天，费用100元上下，检测结果一般当天或3个工作日出具。 （3）三代HIV抗体检测： 窗口期21天，费用在60左右不等，检测结果一般当天或3个工作日出具。

以上就是医院常用的检测方法，有的医院可以选择，有的医院不可以自己选择，具体可咨询医师。

# 自测

2.艾滋病试纸自测

选择自测，则要学会分辨产品，市面上有好的也有坏的，还有恶意引导的，我们要擦亮眼睛购买，自测可分为三大类。

（1） DNA核酸检测 窗口期7天，单次费用1000元左右，窗口期后单次准确率100%，需要自行邮寄血液样本至检测机构，因为需要邮寄所以出结果时间4-7天不等。 （2）四代P24抗原抗体检测试纸 窗口期14天，单次费用180元左右，窗口期后单次准确率99.97%，多份检测阴性则可认定100%，检测方法为取指尖血血检，20分钟出结果，目前国产试纸无四代抗原产品。

（3）三代HIV抗体检测试纸 窗口期28-42天，单次费用10-300元之间。窗口期后单次准确率99%，多份检测阴性则可认定100%，10-20分钟出结果，检测方法目前有血检、唾检、尿检，血检是最准确的方法。

在选择试纸时，尽量选择抗原检测，购买产品时要注意是三代抗体检测产品还是四代抗原产品，避免被商家误导。

# 艾滋病的传播

感染HIV病毒的途径主要有三条:  - 性  - 血液  - 母婴

HIV暴露在空气中之后会在几秒钟到几分钟之内全部死亡

HIV主要存在于传染源的血液、精液、阴道分泌物、胸腹水、脑脊液、羊水和乳汁等体液中。

一些体液，如唾液、汗液和泪液，不会携带HIV

# 性

目前经性传播是艾滋病的主要传播途径，约占95% 国家卫健委： - 经济观察网 － 专业财经新闻网站 (eeo.com.cn) (opens new window) 2017年我国报告感染者中异性传播为69.6%，男性同性传播为25.5%。 在青年群体中，同性比例高得多 男女病人比例约为4:1

无论是同性，还是异性之间的性接触都会导致艾滋病病毒传播。艾滋病病毒感染者的精液或阴道分泌物中有大量的病毒，在性活动时，由于性交部位的摩擦，很容易造成生殖器黏膜的细微破损，这时病毒就会乘虚而入，进入未感染者的血液中。

因此，在性行为过程中一定要使用==安全套==

  • 安全套能防艾滋病吗？ - 知乎 (zhihu.com) (opens new window) 安全套是否一定能防止艾滋病？ 预防概率在96%以上，虽然HIV直径小于安全套的空隙，但是HIV病毒离开了精液是无法存活的，所以不用担心。

注意一定要使用，并规范使用安全套！

关于“口交”，“接吻”，是否会传播HIV？ 理论上，唾液不携带HIV，但是人体口腔很容易出现口腔溃疡，牙龈出血等小伤口，因此还是存在一定风险。

0相对于1，被传染的风险更大 因为作为被插入方，直肠作为排泄器官，弹性远不如阴道，也不会分泌润滑液，因此更容易出现撕裂等微小伤口，增加感染风险。

# 血液

血液传播是感染最直接的途径。输入被病毒污染的血液，使用了被血液污染而又未经严格消毒的注射器、针灸针、拔牙工具，都是十分危险的

# 母婴

如果母亲是艾滋病病毒感染者，那么她很有可能会在怀孕、分娩过程或是通过母乳喂养使她的孩子受到感染。

艾滋病患者是否可以生孩子？ 现在我国有许多“母婴阻断”的成功案例

# 不会传播的场景

一些正常的场景是不会传播艾滋病的

  • 游泳
  • 浴缸
  • 谈话
  • 咳嗽
  • 打喷嚏
  • 马桶
  • 握手
  • 拥抱
  • 吃饭
  • 贴面礼
  • 蚊虫叮咬
  • 接吻（理论上）

蚊虫叮咬为什么不会传播艾滋病 蚊虫叮咬一个人的时候，它们并不会将自己或者前面那个被吸过血的人血液注入。它们只会将自己的唾液注入，这样可以防止此人的血液发生自然凝固。它们的唾液中并没有艾滋病病毒。而且喙器上仅沾有极少量的血，病毒的数量极少，不足以令下一个被叮者受到感染。而且艾滋病病毒在昆虫体内只会生存很短的时间，不会在昆虫体内不断繁殖。昆虫本身也不会得艾滋病。

# 艾滋病的现状

据世界卫生组织统计，截至2021年，全世界约有3840万人携带艾滋病毒，2021年度新增感染者约150万人、死亡约65万人，其中，有约2,060万名HIV感染者生活在非洲东部和南部 自艾滋病毒流行以来截至2021年12月，已有约8,420万人感染艾滋病毒，约4,010万人死于艾滋病相关疾病。艾滋病是人类历史上致死人数最多的流行病之一。

# 中国

截至2018年9月底，全国已报告存活感染者85万（预计总共125万），死亡26.2万例

每年增长约8万

# 艾滋病的预防和阻断

# 高危性行为

容易引起HIV感染的行为统称为艾滋病高危行为，但HIV传染的关键是任何带有足量病毒的体液交换。也就是说，得有足量的病毒进入到人体体内，才可能发生传染。

注意必要条件是：足量，病毒，体液交换。

# 事前 PREP

暴露前预防，其原理也是抑制HIV在人体内的复制。若持续服用PrEP来预防HIV，则血液中此药物成分可阻止病毒进展到感染确立阶段以及在人体内扩散。

PrEP 可将通过性行为感染艾滋病的风险降低约 99%

HIV暴露前预防投药，亦作PrEP，英语Pre-Exposure Prophylaxis，是未感染HIV的人群利用药物对感染进行预防的手段。PrEP的药物成分是特鲁瓦达(Truvada)，是唯一通过美国FDA批准的作为HIV预防药物使用的成分。

PrEP并不是疫苗，身体并不会因为使用PrEP而产生艾滋病病毒抗体，所以需要定期口服使用才有其预防效用。

很多人非常在意的一个问题，服用后我能不戴套吗？

理论上来说，正确服药的前提下PrEP能极高的预防HIV的感染，但它并不预防其它性病，如淋病，衣原体，尖锐湿疣（HPV），梅毒。

关于服用方法的争议：

方法一：每天服用一粒，服用一周后可以达到预防效果，之后仍需坚持每天服药才有预防效果。

方法二：性行为之前2-24小时服用2粒，服用24小时和48小时之后再分别服用一粒，即总共4粒。如果这中间还有其它性行为，最后一次性行为之后还需服用两天。一周不超过7粒。

方法二：只对男男性行为有效，对男女性行为无效

# 事后 PEP

在高危性行为之后，可以通过服用阻断药物来防止感染HIV。

服药时间：

不能低于28天。

阻断作用有效时间：

高危行为后，1～2 小时内服用药物，效果最佳。
72 小时内服用，阻断依然有效。
超过72小时，艾滋病阻断的成功率降低。
超过72小时，阻断药物的效力降低，但是依然可以有效的抑制HIV病毒，对于艾滋病的早期治疗有帮助。

服药后注意事项：

服用2周后，需检查肝肾功能，服药后的1个月和3个月分别抽血检测HIV抗体，确定艾滋病是否阻断成功。3个月时检查结果阴性，可以判定为阻断成功。服药方法不正确时，有极低阻断失败的概率。

副作用：

包括头晕，恶心，乏力，厌食，头疼，多梦，抑郁等，因人而异。

副作用有时会大到影响生活工作的程度，但千万不要因为副作用太大而自行停药。

发生高位之后，请及查询当地阻断医院获取药物 全国HIV暴露后预防门诊点信息一览表 (chinaaids.cn) (opens new window)
