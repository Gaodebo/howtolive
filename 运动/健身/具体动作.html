# 高位下拉

  • 快速下拉，慢速回放
  • 背部直立
  • 不要靠身体向后靠，来挑战大重量

# 坐姿划船

  • 身体直立

不要靠身体前倾或后倾

# 引体向上

  • 向上用力，向下缓慢
  • 向上吐气
  • 主要刺激背部肌肉
  • 不要靠身体惯性向上，来追求数量

# 腿举

  • 背部不要离开靠背
  • 如果靠背可以调整尽可能低
  • 脚蹬中间，训练股四头肌
  • 脚蹬上面，训练股二头肌和臀部
  • 脚蹬下面，损害腰部
