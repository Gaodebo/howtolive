# 如何选购羽绒服

一般看以下几个指标

# 充绒量

充绒量就是羽绒的含量，羽绒越多，越保暖，价格越贵 薄款一般在100g以下，普通在100-200g

# 绒种

一般分为：

  • 鸭绒
  • 鹅绒 相比之下，鸭子生长周期比鹅短，鸭绒保暖效果略差（差距不明显，越5%），所以鸭绒的价格一般都比鹅绒便宜，但是已经够用

此外还有显色之分，白色较少，多用于浅色衣服，但价格也会略高

# 绒子含量

首先，我们来了解以下羽绒服为什么保暖？ 材料的保暖程度是由导热系数决定，最好的是静止空气，其次是羽绒 但是不是羽毛，而是绒子，绒子越多越能锁住更多的空气，就越保暖

新国标规定，50%以上的绒子含量，才能叫羽绒服 一般最好购买90%以上的

# 蓬松度

这个不是必须的，一般蓬松度越高，羽绒服的空间更多，更轻便，回弹性更好

# 清洁度

羽绒服一般是不经常清洗的，可以防水防污

# 如何清洗羽绒服

  • 最好是不要经常清洗
  • 手洗，不要干洗
  • 不要熨烫，漂白
  • 悬挂晾晒，轻轻拍打使羽绒蓬松，不要暴晒
  • 水温不超过30度
  • 建议使用中性洗涤剂，不要使用柔顺剂，浸泡时间不要超过10分钟
