# 一、天然纤维

# （一）棉

# 特性

分类：

细绒棉（陆生棉） 长度28mm 新疆90%产量

长绒棉（海盗棉） 长度28+8mm 新疆10%产量 美国的种植量大 注册商标匹马棉

（加工艺）精梳棉 梳走棉花短毛更光洁平整

（加工艺）丝光棉 棉花泡浓烧碱中剧烈吸水膨胀 有蚕丝光泽

彩棉 棉花基因突变 白色变彩色（棕色、绿色）品质一般

# 注意事项

# （二）麻

# 特性

质感粗糙扎人

# 注意事项

# （三）蚕丝

# 特性

桑蚕丝白色 西南地区生产 做衣服

柞蚕丝黄色 辽宁生产 做被子 价格便宜

双宫茧 做蚕丝被抽不出丝纺织可以做被子

绢丝 蚕丝边角料断蚕丝

# 注意事项

# （四）羊毛

# 特性

绵羊 澳大利亚供应全球40%的羊毛

# 注意事项

# （五）羊绒

# 特性

（开司米=克什米尔）

山羊 中国供应全球80%的羊绒 内蒙新疆主要产地

# 注意事项

# 二、化学纤维

# （一）再生纤维素纤维

天然纤维再经过一道工序

# 1.粘胶 / 竹纤维

# 特性

（所谓的竹纤维）非常不环保

二硫化碳（污染很大）溶解了短棉花 结构疏松了 吸湿性比棉花高50% 湿水易皱

# 注意事项

# 2.莫代尔

# 特性

二硫化碳（污染很大）溶解了短棉花再过一遍锌夜 比粘胶更有韧性不易皱有弹性

适合贴身穿如内衣内裤

# 注意事项

# 3.莱赛尔

# 特性

NMMO溶剂（环保）溶解了短棉花

天丝是奥地利兰精纺丝厂注册的商标，商标中包括莫代尔和莱赛尔

# 注意事项

# 4.醋酯纤维

# 特性

（香烟滤嘴）

醋酐溶解棉花木头 不易吸水 有塑料质感蚕丝光泽 做衣服悬垂性好

# 注意事项

# （二）合成纤维

# 1.涤纶 / 聚酯纤维

# 特性

（的确良）PET

最普通的聚酯纤维 可以做夏凉被、羊羔绒、法兰绒

本身不吸湿 但穿起来可以吸湿排汗

# 注意事项

# 2.锦纶 / 尼龙

# 特性

（尼龙）

辽宁锦西化纤厂第一个生产

耐磨性好 做冲锋衣、羽绒服、防嗮服

价格比涤纶贵三倍

# 注意事项
  • 水温不宜超过45度
  • 忌暴晒和烘干

# 3.腈纶

经jing

# 特性

（外号人造羊毛）

冬天毛衣保暖衣面料

德绒是德国拜耳生产的腈纶商标

美雅碧是日本三菱生产的腈纶商标

色拉姆是日本东洋生产的腈纶商标

腈纶本身不会发热，混纺的粘胶纤维会吸湿发热 保暖要看厚度

# 注意事项
  • 可晾晒

# 4.氨纶 / 弹性纤维

# 特性

有弹力的纤维

混纺到各种面料里增加弹性（加百分之几）

越多弹力越持久

莱卡是杜邦公司生产的氨纶

# 注意事项
  • 不能用84漂白 丧失弹力
